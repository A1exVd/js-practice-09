
**О безопасности: почему мы добавляем node_modules в .gitignore и не отправляем её на GitHub?**
Для того, чтобы не загружать тяжеловесную папку с библиотеками на GitHub. Вместо этого разработчик может установить все необходимое используя package.json и команду npm install.
**О версиях: что значит символ ^ в версии пакета (например, "faker": "^8.4.1")?**
^ символ означает совместимую версию пакета. Разрешает установку до последней патч версии или минорной. Например, ^1.2.5, 1.9.9 - можно, 2.0.0 - нельзя. 
**О выживании: если ты скачал проект из интернета без папки node_modules, какой командой ты его «оживишь»?**
Одной из команд: npm install, yarn install, pnpm install
